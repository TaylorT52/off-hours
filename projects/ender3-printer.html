Customizing an Ender 3 printer.

In progress

Project overview

Ender 3 printers are common open-source machines, but they are still fairly manual and a little rough around the edges. There is no automatic bed leveling by default, setup can be finicky, and getting consistently good prints usually takes iteration.

The goal of this page is to track upgrades, fixes, calibration notes, and what changes actually make the printer better over time.

What I’m tracking

  • printer upgrades and quality-of-life improvements
  • better calibration and more reliable print settings
  • experiments with materials, parts, and workflows

Project Log

Weekly updates.

Week of 07/22

Starting point

First step was to get the printer running. Unfortunately, with a 7 year old second-hand printer, something's bound to go wrong, and a bearing for loading the PLA into the extruder broke off. So, I ordered a replacement piece so I can start printing! Next steps - print, automatic bed leveling, and setting up a Raspberry Pi server so I don't have to load images manually with an SD card.
